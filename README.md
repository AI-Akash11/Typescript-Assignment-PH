## B7A1 - Advanced Problem Solving with TypeScript & OOP

## Overview

This repository contains my solutions for the **B7A1 Assignment: Advanced Problem Solving with TypeScript & OOP**.

The assignment demonstrates fundamental TypeScript concepts, including:

- Functions
- Arrays
- Union Types
- Type Guards
- Interfaces
- Generics
- Object-Oriented Programming (OOP)
- Class Inheritance

## Problems Solved

### Problem 1

Filter even numbers from an array.

### Problem 2

Reverse a string.

### Problem 3

Determine whether the input is a string or number using type guards.

### Problem 4

Retrieve an object's property using a generic function.

### Problem 5

Add an `isRead` property to a Book object.

### Problem 6

Implement inheritance using `Person` and `Student` classes.

### Problem 7

Find the intersection of two arrays.
